# Rasputin

> Open-source homelab cluster system: Raspberry Pi and Intel N100 nodes plus a dedicated
> firewall node, managed from one web UI. Flash a card, boot, open http://rasputin.local.
> Pre-alpha, AGPL-3.0 — image layouts and update formats still change without notice.

## Install

- One command (macOS/Linux): `curl -fsSL {{ "bootstrap.sh" | absURL }} | sudo bash`
- [bootstrap.sh]({{ "bootstrap.sh" | absURL }}): fully scriptable — non-interactive via `RASPUTIN_*` env vars (`RASPUTIN_DRY_RUN=1` for a no-write preflight); see the agents doc above.
- [Download page]({{ "download/" | absURL }}): images, manual flashing steps, verification.
